📖 Cara Pakai ReelGen
🔑 STEP 0 — Siapkan API Key Gemini
1. Buka aistudio.google.com/apikey
2. Login akun Google → klik Create API Key
3. Copy key (dimulai dengan AIza...)
4. Paste di kolom Gemini API Key → tap Simpan
⚠️ Jika muncul "exceeded quota" — tunggu besok atau buat API key baru di project berbeda. Gemini gratis 1500x/hari per project.
📦 STEP 1 — Isi Info Produk
1. Isi Nama Produk (wajib)
2. Pilih Kategori & isi Harga
3. Paste Artikel/Deskripsi produk di kotak besar (wajib) — bisa copy dari toko online
4. Upload 1–6 Foto Produk (tap kotak foto)
✨ STEP 2 — Generate Scene AI
1. Tap tombol ✨ AI di pojok kanan atas
2. Tunggu loading (~10 detik)
3. Pindah ke tab Scene — 8 scene otomatis tersedia
4. Tap tiap scene untuk edit teks/warna/layout
▶ STEP 3 — Build & Preview
1. Tap tombol ▶ Build
2. Otomatis pindah ke tab Preview
3. Tap ▶ PLAY untuk lihat animasi reel
4. Tap layar HP untuk next scene
5. Scroll bawah → tap ⬇ Download WEBM
💡 Tips Scroll di HP
• Geser layar dari area konten (bukan di input/textarea)
• Di textarea artikel: tap dulu → geser dari luar textarea untuk scroll halaman
• Tab Preview: scroll dari bawah foto HP mockup ke bawah
